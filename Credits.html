Tools and Technologies

Tools and Technologies

This project was developed using open-source tools for spatial analysis, web development, and data visualization:

  • QGIS – Spatial analysis and map preparation
  • GeoServer – Serving geospatial data through WMS
  • OpenLayers – WebGIS integration
  • Python – Zonal statistics and data preprocessing
  • HTML, CSS, JavaScript – Website development
  • GitHub – Version control and project collaboration
